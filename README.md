# PI2Atividade5
atividade 5 da PI 2 

Título do projeto: Gestão de vendas online


Status do projeto:em desenvolvimento 

Tecnologias que serão aplicadas: Java e MySQL

Time de desenvolvedores: Marcela e tutores

Objetivo do software:  organizar e gerir as vendas online 



 
Funcionalidades do sistema:

Requisitos funcionais:

•	RF001 – Cadastro de Produtos (Perfil: Gerente)
 Pessoa consegue incluir, alterar ou excluir item. Deve ser preenchido: Nome, fabricante, fornecedor e plataforma.

•	RF002 – Cadastro de Fornecedores (Perfil: Gerente)
 Pessoa consegue incluir, alterar ou excluir dados de fornecedores. Deve ser preenchido: Nome e telefone de contato.

•	RF003 – Cadastro de Vendas (Perfil: Gerente)
Pessoa consegue incluir, alterar ou excluir dados de vendas realizadas. Deve ser preenchido: produto, valor, dia, plataforma. 

•	RF004 – Cadastro de Plataformas (Perfil: Gerente)
Pessoa consegue incluir, alterar ou excluir dados de plataformas usadas. Deve ser preenchido: nome e link. 

•	RF005 – Relatório de vendas (Perfil: Gerente e usuário)
Pessoa consegue trazer dados de vendas aplicando filtros de produtos, plataformas e valores, conforme selecionado, para um intervalo de datas.

Requisitos não funcionais

•	RNF001 – Segurança
Acesso de todos usuários somente com login no sistema. 

•	RNF002 – Conexão
O sistema deverá funcionar sem o uso da internet.
